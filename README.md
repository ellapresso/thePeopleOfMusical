# ThePOM Frontend

ThePOM 프론트엔드 애플리케이션

## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- React 18

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```

## 환경 변수

`.env.local` 파일을 생성하고 다음 변수를 설정하세요:

```env
NEXT_PUBLIC_API_URL=http://localhost:3000/api/v1
```

## 기능

- 단원 이름 검색
- 단원 목록 조회
- 단원 상세 정보 표시
